# NTPE - Novel Translator Professional Edition

Version: 0.1.0 Alpha

## 專案位置

請解壓到：

```text
D:\Python\NTPE\
```

## 第一次測試

```bat
cd /d D:\Python\NTPE
pip install -r requirements.txt
python launcher.py
```

成功時會看到：

```text
NTPE 0.1.0 Alpha - Foundation OK
```

## 目前內容

這是正式 Foundation 版本，只負責確認：

- 專案骨架存在
- 設定檔可以自動建立
- 日誌系統可以啟動
- NVIDIA / Gemini RPM 預設值正確

下一版才會加入真正翻譯引擎。
